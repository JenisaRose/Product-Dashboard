# Product Dashboard

A Full Stack Product & Client Management Dashboard built using React, Node.js, Express, MongoDB Atlas, and Material UI.

## Features

### Product Management
- Add Product
- Edit Product
- Delete Product
- Search Products
- Filter by Category
- Filter by Status
- Status Chips

### Client Management
- Add Client
- Edit Client
- Delete Client
- Search Clients
- Email Validation
- Status Management
- Internal Notes

## Tech Stack

### Frontend
- React (Vite)
- Material UI
- Axios
- React Hot Toast
- Lucide React

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

## Installation

### Clone the repository

```bash
git clone https://github.com/JenisaRose/Product-Dashboard.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
